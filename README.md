# Confusion-Mapper
Every wrong answer a student gives reveals something different about how they think — but most educational software only sees "wrong." ConfusionMapper classifies MCQ distractors into four cognitive error types from the Confusion Fingerprint taxonomy: Recall Failure (RF), Partial Knowledge (PK), Confabulation (CF), and Interference (INT). A human researcher and ChatGPT independently classify each distractor, and the tool computes Cohen's Kappa (κ) — a graduate-level statistical measure of inter-rater agreement — visualized as a semicircular speedometer gauge alongside a 4×4 confusion matrix and per-type agreement chart. I built this as the pre-data-collection quality gate for a real pre-registered randomized controlled trial (n=108 students) I am running at IIT Kanpur's Cognitive Science department under Prof. K. M. Sharika. The tool needed to exist before any student data could be collected — the study protocol requires κ ≥ 0.70 before proceeding. Code in Place gave me the skills to build it. The tool runs fully offline using a built-in NCERT question bank, or generates questions live on any topic via the OpenAI API. Sessions are saved as JSON for research records. Built with Python, tkinter, and the OpenAI API.

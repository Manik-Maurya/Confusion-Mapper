# Confusion Fingerprint Taxonomy

## Recall Failure (RF)

Student lacks a retrievable memory trace.

Example:
- Random answer
- No conceptual connection

---

## Partial Knowledge (PK)

Student possesses incomplete understanding.

Example:
- Correct direction
- Missing critical detail

---

## Confabulation (CF)

Student holds a coherent but incorrect mental model.

Example:
- Classic misconception
- High confidence wrong answer

---

## Interference (INT)

Knowledge from another topic is activated incorrectly.

Example:
- Correct answer for the wrong concept

# The Confusion Fingerprint taxonomy

The Confusion Fingerprint Index sorts wrong answers on a multiple-choice question into four cognitive categories. They are meant to be mutually exclusive: every wrong answer should fit one of the four.

## Recall Failure (RF)

The student has no retrievable memory trace for the topic. The answer looks random, or the student explicitly says they don't know.

Example: answering "mitochondria" to a photosynthesis question, with no logical link.

## Partial Knowledge (PK)

The student is almost right. The direction is correct but the mental model is incomplete. Scaffolding usually resolves this quickly.

Example: knowing photosynthesis needs sunlight, but not that it also produces oxygen.

## Confabulation (CF)

The student holds a coherent but incorrect mental model and selects with confidence. This is the classic textbook misconception.

Example: believing plants get their food from soil because "roots absorb nutrients."

## Interference (INT)

The answer is correct, but for a different topic. Well-encoded knowledge from one area fires in response to a question from another.

Example: answering "chloroplast" (the right organelle for photosynthesis) to a cellular-respiration question.
